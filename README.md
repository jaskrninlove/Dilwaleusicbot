## ⚡ About Dilwale Music Bot

**Dilwale Music Bot** is a powerful Telegram music streaming bot designed to deliver high-quality audio playback, fast response, and smooth queue management for group voice chats.

---

## 🛠 YouTube Streaming Fix (VPS Issue)

YouTube may block VPS IPs. We provide two solutions:

### 1️⃣ API Mode (Recommended)
- Stream audio & video using secure API
- No cookie setup required
- Stable & fast playback

### 2️⃣ Cookies Mode (Fallback)
- Use locally generated YouTube cookies
- Place inside `cookies/` folder
- Helps bypass restrictions if API is not used

## 🛠 Quick Setup

### ✔ Requirements
- Python 3.10+
- ffmpeg
- deno

---

### 📦 Installation

```bash
sudo apt-get update && sudo apt-get upgrade -y
sudo apt-get install python3-pip ffmpeg -y
curl -fsSL https://deno.land/install.sh | sh
git clone https://github.com/jaskrninlove/Dilwalemusicbot.git
cd Dilwalemusicbot
pip3 install -r requirements.txt
cp sample.env .env
vi .env
bash start
```
